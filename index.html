YourJS

A JS library you can use in YOUR chosen namespace.

View on GitHub Download .zip Download .tar.gz

YourJS is a lightweight client-side JavaScript (JS) library. YourJS is not intended to compete with or be a replacement for other JS libraries. It is designed to work well side-by-side with other JS libraries such as jQuery, Ext-JS, etc.

Features

  • Lightweight.
  • Low memory footprint.
  • Well documented.
  • Namespace switching.
  • Logging system (future).
  • Class system (future).
